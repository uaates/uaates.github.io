Python’da Veritabanı Yönetimi

SQLite ve SQLAlchemy ile temel veritabanı işlemleri CRUD · ORM · Model Yapısı

← Blog Ana Sayfa

Python, veritabanı yönetimi konusunda oldukça güçlü ve esnek araçlara sahiptir. Küçük projelerden büyük ölçekli uygulamalara kadar farklı ihtiyaçlara uygun çözümler sunar.

1️⃣ SQLite Nedir?

SQLite, dosya tabanlı, hafif ve kurulumsuz bir veritabanı sistemidir. Küçük ve orta ölçekli projeler için idealdir.

2️⃣ SQLite ile Veritabanı Bağlantısı

Python’da SQLite kullanarak veritabanı bağlantısı kurmak oldukça basittir. 

import sqlite3

connection = sqlite3.connect("database.db")
cursor = connection.cursor()

Bu kod, database.db adlı bir veritabanı dosyası oluşturur ve bağlantıyı başlatır.

3️⃣ SQLAlchemy Nedir?

SQLAlchemy, Python için geliştirilmiş güçlü bir ORM (Object Relational Mapping) kütüphanesidir. SQL yazmadan Python sınıfları üzerinden veritabanı işlemleri yapmanı sağlar.

4️⃣ SQLAlchemy ile Model Oluşturma

Basit bir kullanıcı modeli örneği: 

from sqlalchemy import Column, Integer, String, create_engine
from sqlalchemy.ext.declarative import declarative_base

Base = declarative_base()

class User(Base):
    __tablename__ = "users"

    id = Column(Integer, primary_key=True)
    name = Column(String)
    email = Column(String)

5️⃣ Veritabanı ve Tablo Oluşturma

Model tanımlandıktan sonra tabloyu oluşturmak için: 

engine = create_engine("sqlite:///database.db")
Base.metadata.create_all(engine)

6️⃣ CRUD İşlemleri

SQLAlchemy ile temel CRUD işlemleri:

  • Create: Yeni kayıt ekleme
  • Read: Kayıtları listeleme
  • Update: Kayıt güncelleme
  • Delete: Kayıt silme

7️⃣ Sonuç

Python’da veritabanı yönetimi, doğru araçlar kullanıldığında oldukça temiz ve sürdürülebilir bir hale gelir. Küçük projelerde SQLite, büyük projelerde ise SQLAlchemy gibi ORM’ler kullanmak en doğru yaklaşımdır.
