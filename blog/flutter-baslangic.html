Flutter ile API Entegrasyonu

REST API, HTTP paketleri ve JSON veri yönetimi GET · POST · Async · JSON

← Blog Ana Sayfa

Flutter uygulamalarında API entegrasyonu, modern mobil uygulamaların temel yapı taşlarından biridir. REST API’ler aracılığıyla sunucudan veri alabilir, veri gönderebilir ve uygulamanı dinamik hale getirebilirsin.

1️⃣ HTTP Paketi Nedir?

Flutter’da API istekleri yapmak için en yaygın kullanılan paket http paketidir. Bu paket ile GET, POST, PUT, DELETE gibi HTTP isteklerini kolayca gerçekleştirebilirsin. 

dependencies:
  http: ^0.13.3

2️⃣ API’den Veri Çekme (GET)

Bir REST API’den veri almak için http.get() metodu kullanılır. 

import 'package:http/http.dart' as http;
import 'dart:convert';

Future fetchData() async {
  final response = await http.get(
    Uri.parse('https://jsonplaceholder.typicode.com/posts')
  );

  if (response.statusCode == 200) {
    final data = jsonDecode(response.body);
    print(data);
  } else {
    throw Exception('Veri alınamadı');
  }
}

3️⃣ JSON Veriyi Parse Etme

API’den gelen veriler genellikle JSON formatındadır. Bu verileri model sınıflarına dönüştürmek, uygulamanın daha düzenli olmasını sağlar. 

class Post {
  final int id;
  final String title;

  Post({required this.id, required this.title});

  factory Post.fromJson(Map json) {
    return Post(
      id: json['id'],
      title: json['title'],
    );
  }
}

4️⃣ FutureBuilder Kullanımı

API’den gelen verileri ekranda göstermek için FutureBuilder kullanılır. 

FutureBuilder(
  future: fetchData(),
  builder: (context, snapshot) {
    if (snapshot.connectionState == ConnectionState.waiting) {
      return CircularProgressIndicator();
    } else if (snapshot.hasError) {
      return Text('Hata oluştu');
    } else {
      return Text('Veri yüklendi');
    }
  },
)

5️⃣ En İyi Uygulamalar

  • API çağrılarını servis sınıflarında topla
  • Model–View–Service ayrımı yap
  • Hata durumlarını kullanıcıya bildir
  • Loading state’leri yönet

6️⃣ Sonuç

Flutter ile API entegrasyonu, uygulamanın profesyonel seviyeye çıkmasını sağlar. Sağlam bir mimari ile yazılmış API katmanı, performans ve sürdürülebilirlik açısından kritiktir.
